# 8パズル（スライドパズル）

ブラウザ上で遊べる 3×3 のスライドパズルです。1〜8 の数字プレートを空きマスへスライドさせて、
左上から右へ小さい順に並べることを目指します。

## 完成形

```
 1  2  3
 4  5  6
 7  8  □
```

## 機能

- **手動モード**: 空きマスに隣接するプレートをクリック / タップで移動
- **シャッフル**: ランダムな正規手を適用して混ぜるため、必ず解ける配置のみ生成
- **自動解答**: A* 探索（マンハッタン距離）で最短手順を計算
  - 1手ずつアニメーション表示（速度スライダー付き）
  - 一瞬で完成形にするモード
- **手数カウンター / 1手戻す（アンドゥ）**

## 使い方

`index.html` をブラウザで開くだけで動作します。ビルドや外部ライブラリは不要です。

## ファイル構成

- `index.html` — ゲーム本体（HTML / CSS / JavaScript を1ファイルに集約）
