# 8パズル（スライドパズル）

ブラウザ上で遊べる 3×3 のスライドパズルです。2つのバージョンがあります。

| バージョン | ファイル | プレート表示 | デザイン |
|---|---|---|---|
| 数字版 | `index.html` | 1〜8 | 木製プレート×藍色 |
| ABC版 | `abc.html` | A〜H | ステンレス調メタル×グラファイト |
| カー版 | `car.html` | 1〜8 | ホワイト×レーシングレッド。完成すると赤いロードスターのイラストがフェードイン |

どちらもプレートを空きマスへスライドさせて、左上から右へ順に並べることを目指します。
各ページの下部にあるリンクで、もう一方のバージョンへ切り替えられます。

## 完成形

```
数字版          ABC版
 1  2  3        A  B  C
 4  5  6        D  E  F
 7  8  □        G  H  □
```

## 機能

- **手動モード**: 空きマスに隣接するプレートをクリック / タップで移動
- **シャッフル**: ランダムな正規手を適用して混ぜるため、必ず解ける配置のみ生成
- **自動解答**: A* 探索（マンハッタン距離）で最短手順を計算
  - 1手ずつアニメーション表示（速度スライダー付き）
  - 一瞬で完成形にするモード
- **最短手数の表示**: シャッフルした時点で、その配置を解くのに必要な最短手数を表示
- **手数カウンター / 1手戻す（アンドゥ）**

## 使い方

`index.html`（数字版）または `abc.html`（ABC版）をブラウザで開くだけで動作します。
ビルドや外部ライブラリは不要です。

## ファイル構成

- `index.html` — 数字版（1〜8・木製調）
- `abc.html` — ABC版（A〜H・メタル調）。内部の盤面管理は数字版と同じで、表示だけ A〜H に対応
- `car.html` — カー版（1〜8・ガレージ調）。完成時にオリジナルの車イラスト（SVG）が浮かび上がる
- `solver_test.js` — A* ソルバーの単体検証スクリプト（Node.js 用）
